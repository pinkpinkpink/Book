# 用户命令

- hadoop archive 创建一个hadoop档案文件

- hadoop distcp 递归地拷贝目录和文件
 
- hadoop fs 运行一个文件客户端

- hadoop fsck 运行hdfs文件系统检查文件

- hadoop jar 运行jar文件

- hadoop job mapreduce 交换作业

- hadoop pipes 运行pipes文件

- hadoop CLASSNAME 运行类
